# 🐉 RoleMIAsterへようこそ！

サイコロ（またはマウス）を手に取り、想像力次第でどこまでも広がる冒険に飛び込みましょう！**RoleMIAster** は、AIが駆動する無限のRPGで、あなた自身が物語のヒーロー（または悪役）になれます。

## ⚠️ 法的免責事項（退屈だけど必要な部分）

**読まなければサイコロの怒りに直面するでしょう：**

1.  **🤖 統合ローカルAI:** RoleMIAsterは独自のデジタルブレインを搭載しています。インターネット接続なし、奇妙な設定なしで動作します。インストールしてプレイ！
2.  **☁️ オプションのクラウド:** 外部AI（OpenAI、Geminiなど）にお金をかけたいなら、クレジットカードの請求書は自己責任です。
3.  **🎲 ジェネレーティブカオス:** AIは物語をその場で作り出します。語り手が狂って牛が降ってくると言ったら、私たちを見ないでください。開発者は、モデルが生成する奇行にいかなる責任も負いません。
4.  **🏗️ アーリーアクセス:** これは工事中です。バグを見つけた場合、それは欠陥ではなく、「驚きの機能」です！（フィードバックを送ってください、とても助かります）。

## 🖥️ システム要件とデジタルブレイン（AI）

RoleMIAsterは**ローカルAI**で動作します。つまり、PCが思考の重労働を行います。インターネットは不要ですが、恐竜時代のものではないグラフィックカードが必要です。（注：部屋が寒い場合、このゲームは無料の暖房として機能します）。

### ⚙️ 初心者向け概念

**VRAM（ビデオメモリ）**はAIのガソリンです。
*   RoleMIAsterは、物語の糸口を失わないために、**12,000〜16,000語（トークン）**を「記憶」する必要があります。
*   GPUのメモリが不足すると、ゲームは喘息持ちのカニよりも遅くなります。

### 🚨 助けて！PCが遅い！（VRAMの問題）

ゲームの応答がWhatsAppであなたの片思いよりも遅い場合、おそらくVRAMが不足しています。

**欲張りすぎた場合の症状：**
1.  **モデルがRAMにオフロードされる:** PCがグラフィックカードではなく通常のメモリを使用します。結果：極端な遅延。
2.  **軽度のオーバーフロー（1〜2GB）:** 応答ごとに30〜60秒かかります。
3.  **中程度のオーバーフロー（3〜5GB）:** 2〜5分。コーヒーを買いに行くことができます。
4.  **重度のオーバーフロー:** 10〜30分。プレイ不可能。

#### 🛡️ 緊急プロトコル
うまくいかない場合は、次の手順を順番に実行してください。

1.  **📉 VRAMの割合を下げる:** 「設定」に移動し、**85%**まで下げます。再起動して試してください。
2.  **🧹 クリーンアップ:** 40個のタブを開いているブラウザを閉じます。それらがメモリを食い潰しています。
3.  **🛑 70%の制限:** **70%** VRAM割り当てを下回らないでください。
    *   *なぜ？* AIは「考える」（KVキャッシュ）ためのスペースを必要とします。70%未満を与えると、AIは空白になりエラーが発生します。
4.  **🔙 戦術的撤退:** 何もうまくいかない場合は、より小さなモデルにダウングレードします（7Bから3Bへ）。

### ⚙️ モデルの詳細設定

**設定 → 統合ローカルAI**でボタンをいじりたい人のために：

| オプション         | 機能                                 | 推奨事項                                       |
| :----------------- | :----------------------------------- | :--------------------------------------------- |
| **GPU Layers**     | モデルのどれだけをグラフィックカードにロードするか | **-1** (「すべて」を意味します。何をしているかわからない限り触らないでください) |

| **履歴 RAG**       | 遠い過去の出来事を記憶する             | **はい** (長期キャンペーンには必須)                      |
| **Embeddings キャッシュ** | 記憶検索を高速化する                 | **はい** (より速く動作します)                      |

## 📜 ゲームルール（はい、ルールがあります）

RoleMIAsterは、クラシックで堅牢なd100（100面ダイス）システムを使用します。

### 1. あなたの属性（あなたを定義するもの）
テーブルトップRPGのようにランダムに生成されます！

*   💪 **筋力 (STR):** 強く打つため、石を持ち上げるため。
*   ❤️ **耐久力 (CON):** 衝撃、病気、毒に耐えるため。
*   🐘 **体格 (SIZ):** 大きくて重いか、小さくてすばしっこいか。
*   🏃 **敏捷性 (DEX):** 器用さ、狙い、自分の足につまずかないこと。
*   🧠 **知性 (INT):** 学習、記憶、パズル解決。
*   🔮 **意志力 (POW):** あなたの魂、運、精神力、魔法。
*   😎 **魅力 (CHA):** リーダーシップ、魅力、人々を騙す能力。

### 2. 簡単な計算（派生特性）
*   **体力 (HP):** `(CON + SIZ) / 2`。0になると…まあ、わかりますね。F。
*   **マナ (MP):** POWと同じ。マナがなければ、トリックはありません。（マナのない魔法使いは、コスプレをしているローブを着た男にすぎません）。
*   **アクションポイント (AP):** `(INT + DEX) / 2`。イニシアチブを決定します。最初に殴る者は二度殴る。

### 3. あなたのスキル
基本値（例：隠密 = DEX + INT）で始まり、使用することで向上します。パーセンテージ（%）です。

### 4. アクション解決（ダイス）
危険なこと（登る、嘘をつく、攻撃するなど）を試みると、AIはロールを要求します。
ゲームは不可視の**d100**を振ります。（覚えておいてください：サイコロはあなたを憎んでいます。個人的なことです）。

*   **あなたの目標:** スキル値**以下**を出すこと。
    *   *例:* 登攀スキルの値が45%です。
    *   30を出す：✅ **成功**。サルのように登ります。
    *   80を出す：❌ **失敗**。激しくぶつかります。
    *   01〜05を出す：✨ **クリティカル！** 後方宙返りで壮大なパルクールをします。
    *   99〜100を出す：💀 **クリティカル失敗！** 途中でズボンが落ち、顔から転びます。

## ⚔️ 戦闘（武器を取れ！）

言葉が通用しないとき、戦いの時が来ます。戦闘は**戦術的**で**ティック**（時間）で機能します。

### 🕹️ 仕組み：
*   **ティック:** 各アクションには時間がかかります。重い武器は遅いです（多くのティックを消費します）。短剣は速いです。
*   **あなたの速度:** アクションポイント（AP）に依存します。

### 基本アクション：
*   **🗡️ 攻撃:** 抜いているもので殴りつけます。
*   **🛡️ 防御:** 防御態勢を取ります。ダメージを軽減しますが、スタミナを消費します。
*   **🏃 逃走:** 臆病者（または賢く生きたい人）のために。
*   **🏳️ 降伏:** 勝てないと思ったら、憐れみを乞うことができます。
*   **✨ 魔法/サイバネティクス:** 火、レーザー、ハッキング…

### ⚠️ 生存の黄金律
**戦う前に武器を抜きましょう！**
戦闘中に武器を抜くには時間がかかります（ティック）。鞘に入ったまま捕まると、それを不器用に抜こうとしている間に徹底的に殴られます。または、**さらに悪いことに！** 武器を装備スロットではなく（すみません、バックパックからマンダブを取り出す必要があります、どこかにあったはずですが…どこかに…） **バックパックに入れている場合！** 🤦‍♂️
👉 **（右パネル）**装備されている武器のラベルを**クリック**して抜いてください。緑色に点灯します。

戦闘中に武器を動かそうとすると時間がかかり（ティック）、不意打ち攻撃につながる可能性があり、さらにターンを失う可能性が非常に高いです。斧が額に迫っているプレッシャーで失敗するかもしれません。または、キャラクターにADHDがあるのは良いアイデアではなかったかもしれません。

## ✨ 魔法とサイバネティクス

あなたの世界の「中世ファンタジー」または「サイバーパンク」によって異なります：

### 🧙‍♂️ ファンタジー：エンチャント
あなたは魔術師ですか？装備にエンチャントを施すことができます。
1.  **学習:** グリモワールに呪文が必要です。
2.  **キャスト（ドラッグ＆ドロップ）:** インベントリのアイテムの上に、リストから呪文をドラッグします。
3.  **コスト:** アクティブなエンチャントを維持するには**ターンごとにマナを消費します**。
4.  **警告:** 戦闘中にマナが枯渇すると、炎の剣は錆びた鉄の塊に戻ります。

### 🦾 サイバーパンク：インプラント
ズーム機能付きの目やゴリラの腕が欲しいですか？
1.  **サイバネティック・サージョン（NPCサービス）**を探します。
2.  クレジットを支払います（未来では何も無料ではありません）。
3.  **人間性のコスト:** 各インプラントはあなたの魂の一部（**人間性**）を奪います。
    *   あまりにも多くをインストールすると、**サイバーサイコシス**（殺戮する足付きトースター）になり、キャラクターのコントロールを失います。
